# zScale Capital Website

A modern, performant, SEO-friendly landing page for zScale Capital - the North Texas Startup Ecosystem Hub.

## Overview

This landing page is designed to help Dallas-area founders navigate the local startup ecosystem with tools, data, and mentorship resources.

## Features

- **Modern Design**: Inspired by premium VC websites like Felicis, featuring elegant serif typography (Instrument Serif) paired with clean sans-serif body text (Satoshi)
- **SEO Optimized**: Includes comprehensive meta tags, Open Graph data, and semantic HTML structure
- **Fully Responsive**: Optimized for desktop, tablet, and mobile viewports
- **Performance First**: Single HTML file with embedded CSS and JavaScript for fast loading
- **Smooth Animations**: Scroll-reveal animations and micro-interactions for enhanced UX
- **Accessibility**: Proper ARIA labels, semantic markup, and keyboard navigation support

## Sections

1. **Hero** - Main value proposition with CTA buttons and key stats
2. **Metrics** - North Texas Venture Pulse statistics
3. **Process** - 3-step readiness system (Audit, Optimize, Deploy)
4. **Resources** - Founder's Survival Kit tools and downloads
5. **Advisors** - Dallas Advisor Network showcase
6. **FAQ** - Common questions with accordion UI
7. **Newsletter** - Dallas Startup Weekly signup
8. **Footer** - Navigation links and social media

## Tech Stack

- Pure HTML5/CSS3/JavaScript (no frameworks)
- Google Fonts (Instrument Serif, Satoshi)
- CSS Custom Properties for theming
- Intersection Observer for scroll animations
- Mobile-first responsive design

## Usage

Simply open `index.html` in a web browser, or serve with any static file server:

```bash
# Python
python3 -m http.server 3000

# Node.js
npx serve

# Or deploy to any static hosting (Netlify, Vercel, GitHub Pages, etc.)
```
